# Weather App

Open API를 이용한 날씨 앱입니다.



### Feature

1. 현재 위치 기반의 날씨 정보
2. 도시 검색 기능 및 목록 추가
3. 추가한 도시의 날씨 정보

### Git Management

- master : 최종본  
- develop : 코드리뷰 후 Merge 버전, Hotfix 버전, release 버전  
- feature : 마일스톤 단위로 관리

**Commit Message 양식은 아래의 규칙을 따른다.**  

```
1. feature : [마일스톤 ID] 메세지  
2. develop : [Merge 날짜] 메세지  
3. master : [배포 버전_날짜] 메세지  
```
